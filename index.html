Política de Privacidad

Esta aplicación académica (“Sistema de Control de Proyectores”) utiliza la autenticación de Google únicamente para identificar a los usuarios de la facultad.

Datos recopilados:

  • Nombre, correo electrónico y foto de perfil (solo para mostrar en la interfaz).
  • Acceso al calendario personal para crear y leer eventos relacionados con la solicitud de proyectores.

Uso de los datos:

  • Los datos personales se usan exclusivamente para autenticar al usuario y mostrar su información en la app.
  • El acceso al calendario se usa únicamente para crear eventos de tipo “Solicitud de proyector” y listarlos para gestión.
  • NO se comparten, venden ni almacenan datos fuera de la sesión del usuario.

Seguridad:

La aplicación no almacena tokens de Google ni contraseñas. Todo el acceso al calendario se realiza mediante el token de sesión actual del usuario.

Finalidad:
